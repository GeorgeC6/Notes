# 软离子器件力学：从原理到应用

## 离子导体

载流子：离子

- 高可拉伸性
- 高透明度
    - 水凝胶
    - ITO（氧化铟锡）
    - 碳纳米管（CNT）
    - 银纳米线（AgNW）
    - 石墨烯（Graphene）
- 较小的电阻-应变依赖性

长方体，初始长度 $l_0$，横截面积 $A_0$，初始电阻 $R_0 = \rho \frac{l_0}{A_0}$

拉伸到长度 $l$，横截面积 $A$. 由 $A_0 l_0 = A l$，记拉伸比 $\lambda = \frac{l}{l_0} = \frac{A_0}{A}$，则

$$
R = \rho \frac{l}{A} = \rho \frac{\lambda l_0}{A_0 / \lambda} = \lambda^2 R_0.
$$

## 水凝胶

> 最常用 PAAm hydrogel: 聚丙烯酰胺水凝胶
>
> 制备：
> 
> - 单体（momomer）：丙烯酰胺（AAm）
> - 交联剂（crosslinker）：N,N'-亚甲基双丙烯酰胺（MBAA）
> - 引发剂（initiator）：过硫酸铵（APS）
>
> 引发剂把单体和交联剂的双键打开

龚：双网络水凝胶
