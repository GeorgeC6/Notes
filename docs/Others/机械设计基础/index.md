!!! summary "考试重点"
    判断、选择 60%（题库），计算、作图 40%

    - 自由度计算（虚约束）

    - 轮系传动比，周转轮系/定轴轮系（有公转的就是行星轮 -> 行星架 -> 中心轮）

    - 蜗轮蜗杆、斜齿轮、受力判断

    - 螺旋传动，位移速度关系

    - 连杆设计**作图**，概念：极限位置、摆角、行程（曲柄滑块）

    - 凸轮**作图**：基圆，回程，...注意旋转方向

    - 强度计算、校核不考，带传动、链传动不考

    - 改错题
