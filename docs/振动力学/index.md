## 考试范围

1.  单自由度线性系统
    - 谐和激励（幅频、相频特性，频率比 $0, \infty, 1$ 及其物理解释） 
    - 任意激励（脉冲响应/Laplace 变换）
2.  多自由度线性系统
    - 本征频率、振型的求解，列方程
    - 正交性
    - 比例阻尼，方程简化
    - 了解消振器
3.  :star: **连续系统的振动**
    - 列方程，初始条件与边界条件
    - 求解
        - 边界条件齐次化
        - 特征值问题
        - 正交性与展开系数
4.  近似计算方法
    - 动能、势能形式
    - 了解假设模态法
    - 瑞利法
    - 子空间迭代法
        - 逆迭代法
        - 和瑞利法的组合
    - 时间积分法
        - 线性加速度法
        - 了解 Newmark 法
5.  振动控制
    - 可控性
    - 可观性
